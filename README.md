# Test-Time Optimization for Deblurring and Super-Resolution

This repository hosts preliminary materials for an ongoing research project on test-time optimization for image restoration.

## Overview

We explore test-time optimization methods for image restoration tasks, including image deblurring and super-resolution. The goal is to adapt the restoration process to each test image using image-specific degradation and reconstruction constraints.

## Motivation

Image restoration models often suffer from performance drops when test-time degradations differ from those seen during training. Test-time optimization provides a flexible mechanism to improve image-specific restoration without requiring full retraining.

## Method Preview

Our current framework investigates:
- test-time optimization for deblurring
- test-time optimization for super-resolution
- adaptive reconstruction objectives
- structure-aware restoration constraints
- image-specific optimization at inference time

Full technical details will be provided in the paper.
